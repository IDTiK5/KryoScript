# KryoScript

**KryoScript** là một script Roblox hỗ trợ các chức năng ESP, aimbot và nhiều tính năng tùy chỉnh khác. Script này được thiết kế để hoạt động trên hầu hết các trò chơi Roblox (universal), giúp người chơi có trải nghiệm tốt hơn với các công cụ hỗ trợ tối ưu.

## Tính năng nổi bật

- **ESP**: Hiển thị player, đồ vật, vật phẩm... trên màn hình trò chơi.
- **Aimbot**: Tăng độ chính xác khi nhắm mục tiêu.
- Hỗ trợ đa dạng các game Roblox.
- Tùy chỉnh giao diện và tính năng trong script.

## Hướng dẫn sử dụng

1. Tải hoặc copy mã KryoScript vào trình executor Roblox bạn đang sử dụng.
2. Chạy script trong game Roblox bất kỳ.
3. Mở menu (nếu có) để tùy chỉnh các chức năng như ESP hoặc aimbot theo ý muốn.
